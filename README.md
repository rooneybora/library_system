# Library_Management_System
A simple and functional Library Management System Implemented in python 3 (Tkinter module)
# Installation:
1. Clone repository 
2. Run `entrypoint.py' on your project directory.
3. Ensure all dependencies are installed

# Usage:
1. Issue Book
2. Renew Book
3. Return Book
4. Search Student
5. Search Book
6. Add User
7. Remove User
8. Add Student
9. Remove Student
10. Add Book
11. Remove Book

# Brief Introduction

A student can be issued a book for 03 days.A studnet can take a  maximum of three books after which he has to return in order to receive more.
